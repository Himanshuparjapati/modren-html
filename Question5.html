HTML5 feature showcase

1. semantic tag

HTML5 introduce a set of semantic tag that provide a more meaningful structure to your documents.

for example, the <header>, <nav> <article>,<section>tags help to define the different parts of your webpage more clearly.

2.audio and video support

HTML5 includes native support for embedding audio and video tag without the need of third party plugins. the <audio>and <video> tag allow easily integreat multimedia contentinto your web pages

for example, you can use the following code to embed a video:

        <video controls>
            <source src="example.mp4" type="video/mp4">
            Your browser does not support the video tag.
        </video>

3. Form Enhancements

HTML5 introduces new input types and attributes that make form handling more powerful and user-friendly.

For instance, the <input type="email"> and <input type="url"> allow browsers to validate email addresses and URLs without custom JavaScript. Additionally, the <input type="date"> provides a date picker for easier date selection.
